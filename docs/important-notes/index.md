Simplyblock is a high-performance distributed block storage optimized for Kubernetes that is compatible
with any Linux environments. It also provides integrations with other environments, such as Proxmox.

To enable the successful operation of your new simplyblock cluster, this section defines some initial conventions and
terminology when working with this documentation.
